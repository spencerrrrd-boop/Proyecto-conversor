Caso Práctico: Calculadora de Divisas para Viajeros Frecuentes
Contexto:
Camila y Diego son viajeros frecuentes. Cada mes visitan un país distinto por razones laborales y necesitan convertir su dinero según la moneda local del país en el que se encuentren. Para facilitar sus viajes, siente la necesidad de su propia calculadora de divisas.

Su objetivo es poder ingresar su presupuesto y la tasa de cambio del país que visitan, y obtener automáticamente el equivalente en la moneda local.

Tu misión como desarrollador/a:
Ayuda a Camila y Diego a crear una herramienta digital que les permita realizar esta conversión de manera rápida y sencilla.

Instrucciones:
Crea una función en Python llamada exchange_money que reciba dos parámetros:

budget: la cantidad de dinero en la moneda original que desean cambiar.

exchange_rate: la tasa de cambio correspondiente al país donde se encuentren.

La función debe devolver el valor equivalente en la moneda extranjera.

Nota importante para la investigación:
Antes de crear la función, realiza lo siguiente:

Investiga cuál es la fórmula adecuada para convertir una cantidad de dinero de una moneda a otra usando una tasa de cambio.

Consulta fuentes confiables (por ejemplo, Google, sitios de conversión de divisas o bancos) para identificar la tasa de cambio de al menos tres países diferentes (puedes elegir Japón, México, Alemania, Colombia, etc.).

Anota los valores y prueba tu función con esos datos reales.

Ejemplo de uso:
Si están en Japón y desean cambiar 300 USD a yenes, con una tasa de cambio de 0.0075 (es decir, 1 yen = 0.0075 USD):

exchange_money(300, 0.0075)
Resultado esperado:

40000.0
Objetivos del ejercicio:
Desarrollar habilidades para usar variables y crear funciones en Python.

Fomentar la búsqueda activa de información y uso de datos reales (tasas de cambio).

Aplicar razonamiento lógico y matemático para resolver problemas cotidianos.

Valorar cómo la programación puede ser útil en la vida diaria.
